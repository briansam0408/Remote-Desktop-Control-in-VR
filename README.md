## ðŸ§  Project Summary
Traditional remote desktop tools (like AnyDesk or TeamViewer) are limited to 2D screens.  
This project enhances the experience by:
- Streaming a real-time desktop view into a VR space  
- Allowing users to interact naturally using **controllers**, **head tracking**, and **gestures**  
- Providing a **secure**, **low-latency**, and **immersive** workspace for productivity and remote collaboration

## ðŸ§© System Architecture
**Client-Server Model**
- **Host Desktop (Server):** Captures and streams desktop view in real-time  
- **VR Client (Unity):** Renders the streamed view in 3D space and handles user interactions  
- **Networking Layer:** Ensures secure and low-latency streaming (WebRTC)  
- **Input Handling:** Maps VR gestures and controller actions to desktop commands
